vadmode 0 1.0 (必须)验证方式:
0 = 失焦验证 | 1 = 逐个验证 | 2 = 实时验证

tipmode 0 1.0 (必须)提示方式:
0 = 不显示 | 1 = 直接显示 | 2 = 聚焦显示 | expr = 指定一个标签显示提示信息(些时验证方式为逐个验证)
使用: {tipmode:0} 		 
tipTpl $ 1.0 (必须)提示信息的模板:
